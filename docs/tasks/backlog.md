## 候选垂直切片

- 增加保存冲突解决：在普通保存检测到外部修改后提供重新加载、强制覆盖或取消，并保护当前编辑内容。
- 增加未保存关闭保护：关闭标签或窗口时提供保存、放弃或取消；保存失败不得继续关闭。
- 增加多标签会话：每个标签独立维护内容、后端可信文档状态、保存状态与关闭保护。
- 增加代码文本识别与最小语法高亮。
- 增加 Markdown 编辑与预览。
- 增加 Mermaid 编辑与预览。
- 增加其他传统编码与 GB18030 支持。
- 增加签名、安装包、公证和自动更新流程。

`current.md` 中已承诺的另存为/首次保存是冲突解决与关闭保护的前置；二者又是多标签的前置。该关系只用于避免提前构建错误抽象，不代表本文件中的事项已承诺；任何候选进入 `current.md` 前仍需形成对应 Feature Spec 并确认范围。
