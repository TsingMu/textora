# 产品说明

## 定位

Textora 是一款面向 macOS 的桌面文本编辑器，定位类似 Notepad++ 和 UltraEdit，为文本、代码和轻量标记内容提供统一的查看与编辑体验。

## 目标用户

- 在 macOS 上频繁处理本地文本文件的用户
- 需要查看或编辑代码、Markdown、Mermaid 内容的开发者和技术用户

更细分的核心用户与优先场景尚待确认。

## 已确认范围

- macOS 桌面应用
- 普通文本编辑
- 代码文本编辑
- Markdown 支持
- Mermaid 支持

“支持”的具体深度（例如语法高亮、预览、实时渲染或导出）需由各功能规格定义，不能从上述范围自动推导。

## 产品原则

- 文本内容与用户文件安全优先，避免静默丢失或破坏内容。
- 基础编辑体验应保持直接、快速，增强功能不应阻塞普通文本编辑。
- 对文件编码、换行符或格式进行转换时，应让影响可理解且可控。
- 新功能先形成可验证的最小垂直切片，再逐步扩展。

## 非目标

尚未确认长期非目标。任何排除项应在明确产品决策后记录，不根据同类产品能力自行推断。

- 首个可交付版本的核心用户与首要工作流是什么？

- 代码支持需要达到何种程度：仅高亮，还是包含语言服务、补全与诊断？

- 首发支持哪些 macOS 版本、文件编码和换行格式？
