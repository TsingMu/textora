# 产品说明

## 定位

Textora 是一款面向 macOS 与 Windows 的桌面文本编辑器，定位类似 Notepad++ 和 UltraEdit，为文本、代码和轻量标记内容提供统一的查看与编辑体验。

## 目标用户

- 在 macOS 或 Windows 上频繁处理本地文本文件的用户
- 需要查看或编辑代码、Markdown、Mermaid 内容的开发者和技术用户

更细分的核心用户与优先场景尚待确认。

## 已确认范围

- macOS 13+ 桌面应用（Apple Silicon 与 Intel）
- Windows 10 22H2+ 与 Windows 11 桌面应用（x64）
- 普通文本编辑
- 代码文本编辑
- Markdown 支持
- Mermaid 支持
- UTF-8、UTF-8 BOM 与 GBK/Windows CP936 文件
- 首版处理不超过 50 MiB 的单个文件

“支持”的具体深度（例如语法高亮、预览、实时渲染或导出）需由各功能规格定义，不能从上述范围自动推导。

## 产品原则

- 文本内容与用户文件安全优先，避免静默丢失或破坏内容。
- 基础编辑体验应保持直接、快速，增强功能不应阻塞普通文本编辑。
- 对文件编码、换行符或格式进行转换时，应让影响可理解且可控。
- 新功能先形成可验证的最小垂直切片，再逐步扩展。

## 非目标

首版不包含 Windows 7、Windows ARM64、GB18030、GB 级文件、十六进制编辑、LSP、协作或插件市场。

## 开放问题

- 首个可交付版本的核心用户与首要工作流是什么？
- 是否以大文件编辑或项目目录浏览作为早期重点？
- 代码支持需要达到何种程度：仅高亮，还是包含语言服务、补全与诊断？
- Markdown 与 Mermaid 采用编辑、预览、分栏还是其他交互模式？
- 是否以多标签/多窗口或项目目录浏览作为基础编辑之后的重点？
- 首个公开版本采用商店分发还是独立签名安装包？
