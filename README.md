# Textora

Textora 是一款面向 macOS 的文本编辑器，定位类似 Notepad++ 和 UltraEdit，计划支持普通文本、代码、Markdown 和 Mermaid。

## 当前状态

项目处于初始化阶段。目前仓库仅建立了轻量的规格驱动开发（SDD）与项目记忆文档，尚未确定技术栈，也没有可运行的应用。

- `docs/ARCHITECTURE.md`：系统边界、概念模块与架构约束
- `docs/DECISIONS.md`：已接受的重要决策

- `AGENTS.md`：AI 协作与文档维护约定
